OPTION 1 : Live Server (VS Code) ⭐ RECOMMANDÉ
────────────────────────────────────────────────
1. VS Code → Extensions → "Live Server"
2. Ouvrir index.html → Clic droit → "Open with Live Server"
3. Navigateur s'ouvre → http://localhost:5500

✅ AVANTAGE : Refresh automatique, C'est le plus facile!


OPTION 2 : XAMPP
─────────────────
1. XAMPP Control → Start Apache
2. Navigateur → http://localhost/Portfolio/
3. ✅ Portfolio s'affiche


OPTION 3 : Python 3
────────────────────
cd C:\xampp\htdocs\Portfolio
python -m http.server 8000
Navigateur → http://localhost:8000
